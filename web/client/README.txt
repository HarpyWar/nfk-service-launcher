= Веб интерфейс для управления NFK серверами с сайта =

Общается с серверами по API.
proxy.php - все запросы перенаправляются через него на заданный сервер
auth.inc.php - ограничивает доступ по http basic authorization


== НАСТРОЙКА ==

Переименовать config.php.example -> config.php:

	$Users - пользователи, которым разрешен доступ

	$Instances - экземпляры серверных машин, к которым есть доступ по API. Нумерация должна начинаться с 1 и увеличиваться на 1.
